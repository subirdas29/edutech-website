Grid and flexbox.The basic difference between CSS Grid Layout and CSS Flexbox Layout is that flexbox was designed for layout in one dimension - either a row or a column. Grid was designed for two-dimensional layout - rows, and columns at the same time.

Bootstrap and Tailwind. Bootstrap comes with a set of pre-styled responsive, mobile-first components that possess a definite UI kit. Tailwind CSS uses a set of utility classes to create a neat UI with more flexibility and uniqueness. Sites created using Bootstrap follow the generic pattern that makes them look identical.

CSS Box Model. In CSS, the term "box model" is used when talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content are:

01. "Content"-The content of the box, where text and images appear.
02. "Padding"-Clears an area around the content. The padding is transparent.
03."Border"-A border that goes around the padding and content.
04. "Margin"-Clears an area outside the border. The margin is transparent.

Semantic Tag. The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.The following HTML tags can be used to break your page into identified parts:

01. "header" tag defines a header for a web page.
02. "nav" tag It defines a container for navigation links.
03. "section" tag This defines a section in a web page.
04. "article" tag This element contains the main part,
containing information about the web page.
05. "aside" tag The aside content is often placed as a sidebar in a document.
06. "footer" tag It defines a footer for a document or a section.
